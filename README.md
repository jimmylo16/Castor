## Technologies

- Vite
- Material UI
- Typescript
- Firebase
- Jest

## Description

This is a small example of a TODO app, where you can authenticate, and manage your tasks, all with serverless solution!

## Installation

```bash
$ npm install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Test

```bash
# unit tests
$ npm run test

```
